# Функциональные требования: Расчёт повышенной государственной академической стипендии (ПГАС)

### 1. Цель
Автоматизировать процесс оценки достижений студента для определения права на получение ПГАС и расчёта итогового размера выплаты в зависимости от набранных баллов.

### 2. Исходные данные (входные параметры)
Система принимает следующие данные о студенте за последний семестр:

| Параметр | Тип | Диапазон | Описание |
|---|---|---|---|
| **Средний балл за сессию** | вещественное | 0–5 | Среднее арифметическое всех экзаменационных оценок |
| **Наличие академической задолженности** | булево | да / нет | Есть ли несданные экзамены или зачеты |
| **Учебные достижения (баллы)** | целое | 0–100 | Баллы за олимпиады, конкурсы, публикации (внутренняя шкала ВУЗа) |
| **Научные достижения (баллы)** | целое | 0–100 | Баллы за гранты, патенты, научные статьи |
| **Спортивные/культурные достижения (баллы)** | целое | 0–100 | Баллы за спортивные победы или активную культурную деятельность |
| **Нарушения дисциплины** | булево | да / нет | Наличие выговоров, замечаний или иных дисциплинарных взысканий |

### 3. Правила расчёта (бизнес-логика)

#### 3.1. Блок обязательных условий (блокирующие факторы)
Стипендия **НЕ НАЗНАЧАЕТСЯ** (результат — 0 рублей), если выполняется хотя бы одно из условий:
- **Наличие академической задолженности** = «Да» → отказ.
- **Наличие нарушений дисциплины** = «Да» → отказ.
- **Средний балл за сессию** < 4.0 → отказ.

#### 3.2. Расчёт итогового балла
Если все блокирующие условия не выполнены, система рассчитывает **Интегральный показатель достижений (ИПД)** по формуле:

`ИПД = (K_уч * Учебные_баллы) + (K_науч * Научные_баллы) + (K_спорт * Спортивные_баллы)`

где `K_уч + K_науч + K_спорт = 1` (весовые коэффициенты, например: 0.3, 0.5, 0.2).

#### 3.3. Определение размера стипендии
Размер стипендии зависит от рассчитанного `ИПД` и базовой ставки стипендии (`БАЗОВАЯ_СТАВКА` = 2000 руб.):

| Условие | Размер стипендии |
|---|---|
| `ИПД` >= 80 баллов | `БАЗОВАЯ_СТАВКА * 2.0` |
| `ИПД` >= 60 баллов | `БАЗОВАЯ_СТАВКА * 1.5` |
| `ИПД` >= 40 баллов | `БАЗОВАЯ_СТАВКА * 1.2` |
| `ИПД` < 40 баллов | `БАЗОВАЯ_СТАВКА` (обычная стипендия, если нет блокирующих факторов) |

### 4. Примеры расчёта

**Входные данные:**
- `БАЗОВАЯ_СТАВКА` = 2000 руб.
- `K_уч` = 0.3, `K_науч` = 0.5, `K_спорт` = 0.2

| Средний балл | Задолж. | Дисц. | Учебные | Научные | Спорт. | ИПД | Размер стипендии |
| :--- | :---: | :---: | :---: | :---: | :---: | :---: | :---: |
| 4.8 | Нет | Нет | 90 | 95 | 20 | 90.5 | **4000 руб.** |
| 4.2 | Нет | Нет | 70 | 60 | 80 | 67.0 | **3000 руб.** |
| 4.5 | Есть | Нет | 100 | 100 | 100 | — | **0 руб.** |
| 4.9 | Нет | Есть | 100 | 100 | 100 | — | **0 руб.** |
