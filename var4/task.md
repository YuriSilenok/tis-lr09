# Функциональные требования: Расчёт стоимости электроэнергии по трёхзонному тарифу

### 1. Цель
Автоматизировать расчёт ежемесячной платы за электроэнергию для бытового потребителя на основе дифференцированного по времени суток тарифа и прогрессивной шкалы потребления.

### 2. Исходные данные (входные параметры)
Система принимает следующие данные, введенные пользователем или полученные со счетчика:

| Параметр | Тип | Диапазон | Описание |
|---|---|---|---|
| **Потребление в пиковой зоне** | целое | >= 0 | кВт⋅ч, потреблённые с 7:00 до 10:00 и с 17:00 до 21:00 |
| **Потребление в полупиковой зоне** | целое | >= 0 | кВт⋅ч, потреблённые с 10:00 до 17:00 и с 21:00 до 23:00 |
| **Потребление в ночной зоне** | целое | >= 0 | кВт⋅ч, потреблённые с 23:00 до 7:00 |
| **Наличие льготы** | булево | да / нет | Скидка 30% на итоговую сумму (применяется после всех расчётов) |

### 3. Правила расчёта (бизнес-логика)

#### 3.1. Тарифы за 1 кВт⋅ч (руб.)
- **Пиковая зона (T_пик):** 6.50 руб.
- **Полупиковая зона (T_полупик):** 5.00 руб.
- **Ночная зона (T_ночь):** 2.30 руб.

#### 3.2. Прогрессивная шкала (социальная норма)
Государством установлена социальная норма потребления — **150 кВт⋅ч в месяц**. При превышении этой нормы применяется повышающий коэффициент к тарифу.

| Условие | Коэффициент к тарифу |
|---|---|
| Суммарное потребление <= 150 кВт⋅ч | 1.0 (базовый тариф) |
| Суммарное потребление > 150 кВт⋅ч | 1.5 (повышенный тариф) |

#### 3.3. Формула расчёта
1.  **Расчёт суммарного потребления:**
    `Общее_потребление = Пик + Полупик + Ночь`

2.  **Определение итогового тарифа для каждой зоны:**
    `Итоговый_тариф_зоны = Тариф_зоны * Коэффициент`

3.  **Расчёт стоимости по зонам:**
    `Стоимость = (Пик * Итоговый_тариф_пик) + (Полупик * Итоговый_тариф_полупик) + (Ночь * Итоговый_тариф_ночь)`

4.  **Применение льготы (если есть):**
    `Итоговая_стоимость = Стоимость * 0.7`

#### 3.4. Ограничения
-   Результат округляется до двух знаков после запятой.
-   Если `Общее_потребление` = 0, итоговая стоимость равна 0.

### 4. Примеры расчёта

| Пик | Полупик | Ночь | Общее | Льгота | Расчёт стоимости | Итоговая стоимость |
| :--- | :--- | :--- | :--- | :---: | :--- | :--- |
| 50 | 50 | 50 | 150 | Нет | (50*6.50 + 50*5.00 + 50*2.30) * 1.0 = 690 | **690.00 руб.** |
| 70 | 60 | 40 | 170 | Нет | (70*6.50 + 60*5.00 + 40*2.30) * 1.5 = 1201.5 | **1201.50 руб.** |
| 70 | 60 | 40 | 170 | Да | 1201.5 * 0.7 = 841.05 | **841.05 руб.** |
| 100 | 0 | 0 | 100 | Нет | (100*6.50 + 0 + 0) * 1.0 = 650 | **650.00 руб.** |
